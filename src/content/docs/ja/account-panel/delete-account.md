---
title: アカウントの削除
description: TranSFlator アカウントおよび弊社が保持するすべてのデータを永久に削除する方法。GDPR 第 17 条（消去権）に対応。
---

TranSFlator アカウントはいつでも削除できます。削除は取り消し不可能であり、弊社が保持するすべてのデータに波及し、30 日以内に完了します。

## セルフサービスによる削除

1. [panel.transflator.com](https://panel.transflator.com) にサインインします。
2. **設定 → アカウント** を開きます。
3. **アカウントを削除** をクリックします。
4. メールアドレスを再入力して確認します。
5. **永久に削除** をクリックします。

## その後の流れ

Cloud Function が起動し、単一のトランザクションで以下を実行します。

- `users` コレクションからお客様のドキュメントを削除。
- `purchases` コレクション内の `uid == お客様のuid` であるすべての行を削除。
- `translations` コレクション内の `user_id == お客様のuid` であるすべての行を削除。
- Firebase Auth のレコードを削除し、アクティブなセッションを無効化。
- 削除タイムスタンプとユーザーハッシュを含む監査ログエントリを書き込み。

API トークンは即座に無効になります。デスクトップアプリが実行中の場合、次の呼び出しで `401 Unauthorized` となり失敗します。

## 保持されるもの

弊社の制御が及ばない理由により、アカウント削除後も以下の 2 つが保持されます。

- **Stripe の購入記録** — Stripe は、独自の PCI および金融規制上の理由から支払い記録を保持します。必要に応じて Stripe に直接削除をリクエストできます。
- **ポーランドの会計法に必要な請求データ** — お客様のアカウントに対して発行された請求書は、ポーランド会計法第 74 条に基づき 5 年間保持されます。これらにはお名前と支払詳細が含まれますが、翻訳作業に関する内容は含まれません。

## GDPR 第 20 条 — データの書き出し

削除**前**にデータの機械読み取り可能なコピーが必要な場合は、[hello@tucario.com](mailto:hello@tucario.com) までメールをいただければ、30 日以内に JSON エクスポートをお送りします。プロフィール画面にセルフサービスのエクスポートボタンを設置する計画があります。
