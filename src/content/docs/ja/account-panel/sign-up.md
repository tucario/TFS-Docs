[panel.transflator.com](https://panel.transflator.com) のアカウントパネルは、デスクトップアプリがサインインに使用するアカウントを作成する場所です。Firebase をバックエンドとした Flutter Web アプリケーションです。

## サインアップ

1. [panel.transflator.com](https://panel.transflator.com) を開きます。
2. **サインアップ** をクリックします。
3. メールアドレスと強力なパスワードを入力します。
4. **アカウントを作成** をクリックします。

アカウントが Firebase Auth で作成され、対応するドキュメントが Firestore の `users` コレクションに書き込まれます。これにはスターター AI クレジット残高と生成された API トークンが含まれます。

## メールアドレスを確認する

サインアップ直後に、登録したアドレスにワンクリック確認リンクを含むトランザクションメールが送信されます。リンクの有効期限は **48 時間** です。クリックするまで、クレジットの補充やコストのかかる翻訳の実行はできません。

メールが届かない場合は、まず迷惑メールフォルダを確認し、その後パネルで **確認メールを再送** をクリックしてください。

## スタータークレジット

新しいアカウントには、支払うことなく製品を評価できるように、無料のスターター AI クレジット残高がシード（付与）されます。金額と更新頻度はダッシュボードに表示されます。評価期間には有料プランは不要です。

## パスワードのリセット

パスワードを忘れた場合は、サインイン画面の **パスワードを忘れた場合** をクリックしてください。同じトランザクションパイプライン経由で、1 時間有効なリンクを含むリセットメールが送信されます。
