プランの月間割当以上に翻訳したい場合は、いつでも追加クレジットを補充できます。支払いは **Stripe** によって処理されます。

## 補充（トップアップ）の流れ

1. パネルのダッシュボードで **クレジットを購入** をクリックします。
2. リストからパッケージサイズを選択します。
3. **チェックアウト** をクリックします。Stripe がホストする Stripe Checkout が開きます。カード詳細は弊社ではなく Stripe のサーバーで入力されます。
4. 支払いを完了します。
5. Stripe からパネルにリダイレクトされます。バックエンドで Stripe の Webhook が起動し、Firestore に `purchases` ドキュメントを書き込み、`ai_credits` 残高をアトミックに加算します。
6. 数秒以内にダッシュボードで新しい残高が確認できるようになります。

## 保存される情報

完了した購入ごとに、Firestore の `purchases` コレクションに、お客様の `uid`、Stripe セッション ID、Stripe 支払いインテント、支払額、通貨、購入文字数を含むドキュメントが作成されます。カード番号、CVV、その他のカード会員データは**保存しません**。

## 領収書

支払いが完了した時点で、Stripe から TranSFlator アカウントのメールアドレスに領収書メールが送信されます。それが公式な領収書となります。

## 支払いの失敗

支払いが拒否された場合、Stripe の Webhook は起動しないため、購入は記録されず、クレジットも追加されません。弊社側では何も変更されません。別のカードで再度お試しください。

## EU のお客様向けの VAT / 請求書

Stripe はカスタマーポータルを通じて VAT 対応の請求書を発行します。パネルでのセルフサービス請求履歴機能は計画中ですが、それまではアカウントのメールアドレスを添えて [hello@tucario.com](mailto:hello@tucario.com) までご連絡いただければ、PDF をお送りします。
